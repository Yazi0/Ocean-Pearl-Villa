# Ocean Pearl Mirissa

A premium restaurant landing page inspired by the overall structure and coastal vibe of Mila Mirissa, but redesigned as an original React + Tailwind CSS frontend.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Frontend only
- Replace placeholder images/content with real brand assets
- Reservation/contact buttons are ready for future integration
